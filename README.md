# astral-hv-ngfw
Astral HV-NGFW: Free hyperconverged platform with NG firewall, identity, hybrid DNS, hypervisor, VDI/RemoteApp &amp; observability. Python+SQL Server core, Spring Boot UI/orchestration. Modular, auditable, lock-in-free like pfSense. Uses NetworkManager; Python for data/ML.  Network management uses NetworkManager; data collection/ML handled in Python.
